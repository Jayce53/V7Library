Class DatabaseRecord<TRecord>Abstract

Base class encapsulating common CRUD, caching, and eventing logic for tables.

Extend this class to model a specific table. Subclasses configure metadata such as tableName, primaryKey, and derivedFields, then rely on the provided helpers to load rows, hydrate related tables, update/delete, and keep Memcached in sync.

The class coordinates MySQL access via DatabasePool, cache reads/writes via Cache, and cache invalidation notifications using EventBus.
